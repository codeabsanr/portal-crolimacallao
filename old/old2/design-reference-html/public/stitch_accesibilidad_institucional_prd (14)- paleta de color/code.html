Atkinson Hyperlegible

Body Large (20px): Diseñada específicamente para personas con baja visión, diferenciando caracteres similares como I, l, y 1.

Body Default (18px): Este es el tamaño estándar para lectura. Ningún texto debe ser menor a 16px para garantizar el acceso universal.

Captions (16px): Se utiliza para información secundaria o etiquetas de menor jerarquía.

Compromiso de Accesibilidad

  • Contraste: Todos los elementos de texto mantienen una relación de contraste mínima de 7:1 (WCAG AAA) contra su fondo.
  • Objetivos Táctiles: Ningún elemento interactivo es menor a 48x48px para facilitar su uso a personas con movilidad reducida.
  • Navegación por Teclado: Los estados de foco son explícitos con un borde dorado de 3px.
  • Espaciado: Se utiliza una unidad base de 8px para mantener una jerarquía visual consistente y aireada.
